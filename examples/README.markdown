# Weer Usage Examples

1. First, build Weer by `npm start` in the parent directory (`cd ..`).  
   Then run the following commands in this directory.
2. `npm install`
3.
   - `npm run without-bundlers`
   - `npm run webpack`
   - `npm run rollup `
4.
   - Chrome: load unpacked extension from `./dist`
   - Firefox: `cd dist && web-ext run`
